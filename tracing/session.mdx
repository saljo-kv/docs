Session tracking in Netra provides powerful capabilities for monitoring user interactions and application behavior. This section covers how to track sessions, add custom context, and record events within your application.

## Setting Up Session Tracking

Before you can start tracking sessions, initialize the Netra SDK:

<CodeGroup>

```python python
from netra import Netra

# Initialize with default settings
Netra.init(app_name="Your application name")

# Or with custom configuration
api_key = "Your API key"
headers = f"x-api-key={api_key}"
Netra.init(
    app_name="Your application name",
    headers=headers,
    trace_content=True,
    environment="Your Application environment"
)
```

</CodeGroup>

## Session Identification

Set unique identifiers for your sessions:

<CodeGroup>

```python python
# Set session identification
Netra.set_session_id("unique-session-id")
Netra.set_user_id("user-123")
Netra.set_tenant_id("tenant-456")
```

</CodeGroup>

## Custom Context Attributes

Add custom attributes to enrich your session data:

<CodeGroup>

```python python
# Add custom context attributes
Netra.set_custom_attributes(key="customer_tier", value="premium")
Netra.set_custom_attributes(key="region", value="us-east")
Netra.set_custom_attributes(key="plan_type", value="enterprise")
```

</CodeGroup>

## Custom Events

Record important events in your application:

<CodeGroup>

```python python
# Record user feedback event
Netra.set_custom_event(event_name="user_feedback", attributes={
    "rating": 5,
    "comment": "Great response!",
    "timestamp": "2024-01-15T10:30:00Z",
    "category": "positive"
})

# Record conversion event
Netra.set_custom_event(event_name="conversion", attributes={
    "type": "subscription",
    "plan": "premium",
    "value": 99.99,
    "currency": "USD",
    "source": "website",
    "campaign": "spring_sale"
})

# Record error event
Netra.set_custom_event(event_name="error_occurred", attributes={
    "error_type": "authentication",
    "error_message": "Invalid credentials",
    "severity": "high",
    "user_action": "login"
})
```

</CodeGroup>

## Best Practices

1. **Use Descriptive Event Names**
   - Follow a consistent naming convention
   - Use clear, descriptive names that reflect the event's purpose
   - Avoid generic names like "event" or "action"
2. **Include Relevant Attributes**
   - Add attributes that provide meaningful context
   - Include timestamps for time-sensitive events
   - Add user-specific information when relevant
   - Include error details for error events
3. **Maintain Data Consistency**
   - Use consistent attribute names across similar events
   - Maintain consistent data types for attributes
   - Document your event schema if working in a team
4. **Avoid Sensitive Data**
   - Never include PII in event attributes
   - Use appropriate masking or hashing for sensitive values
   - Follow your organization's data protection policies

## Example Scenarios

### E-commerce Checkout Flow

<CodeGroup>

```python python
# Start checkout session
Netra.set_session_id("checkout-123")
Netra.set_user_id("user-456")
Netra.set_custom_attributes("checkout_type", "standard")

# Track cart addition
Netra.set_custom_event("cart_item_added", {
    "product_id": "PROD789",
    "quantity": 2,
    "subtotal": 150.00
})

# Track checkout start
Netra.set_custom_event("checkout_started", {
    "cart_items": 3,
    "total_amount": 350.00,
    "payment_method": "credit_card"
})

# Track payment success
Netra.set_custom_event("payment_successful", {
    "transaction_id": "TXN456",
    "amount": 350.00,
    "currency": "USD",
    "payment_provider": "stripe"
})
```

</CodeGroup>

### User Onboarding Flow

<CodeGroup>

```python python
# Start onboarding session
Netra.set_session_id("onboarding-789")
Netra.set_user_id("user-123")
Netra.set_custom_attributes("onboarding_version", "2.0")

# Track tutorial completion
Netra.set_custom_event("tutorial_completed", {
    "tutorial_id": "basic_setup",
    "duration": 120,  # seconds
    "steps_completed": 5,
    "user_score": 85
})

# Track feature discovery
Netra.set_custom_event("feature_discovered", {
    "feature_name": "advanced_search",
    "discovery_method": "tooltip",
    "user_level": "new"
})
```

</CodeGroup>
