# Netra: AI Application Observability Platform

Netra is a comprehensive platform for AI application observability, designed to provide deep insights into your AI systems while ensuring data privacy and security. The platform consists of:

- **SDKs**: Multi-language libraries for instrumenting your AI applications
- **Dashboard**: Web-based interface for monitoring and analyzing AI system performance
- **Analytics Engine**: Advanced analysis capabilities for AI-specific metrics
- **Security Suite**: Comprehensive protection against AI-specific security threats

## Platform Overview

Netra provides a complete solution for monitoring, analyzing, and securing your AI applications:

### SDKs (Current and Future)

- **Python SDK** (Current): Comprehensive library for Python-based AI applications
- **Future SDKs** (Planned):
  - JavaScript/TypeScript SDK
  - Java SDK
  - Go SDK
  - .NET SDK
  - Rust SDK

Each SDK is designed to work seamlessly with the Netra platform, providing consistent observability across different programming languages and environments.

## Key Capabilities

### Comprehensive AI Observability

Netra SDK offers in-depth monitoring across various components of your AI application:

- **LLM Call Monitoring**: Track interactions with Large Language Models, including prompts, completions, model usage, latency, and errors across multiple providers
- **Vector Database Operations**: Monitor vector database interactions, query performance, data retrieval patterns, and potential bottlenecks
- **HTTP Request Tracing**: Monitor internal and external HTTP communications, providing insights into dependencies and network performance

### Privacy Protection & Security

Netra SDK prioritizes data protection and application security:

- **Advanced PII Detection**: Multiple detection engines including default, Presidio, and custom Regex patterns
- **Configurable Actions**: Options for masking, blocking, or flagging detected PII
- **Comprehensive Coverage**: Detection of various PII types including names, addresses, phone numbers, email addresses, social security numbers, credit card numbers, and medical records
- **Real-time Processing**: Immediate detection and masking of sensitive information
- **Context-aware Detection**: Understanding PII in different contexts and formats
- **Audit Trail**: Maintaining records of PII detection and masking actions
- **Security Scanning**: Proactive prompt injection detection and prevention

### Industry-Standard Integration

Built on OpenTelemetry standard for collecting and exporting tracing and metrics data, ensuring broad compatibility with observability backends.

### Effortless Instrumentation

Netra SDK streamlines the process of adding observability to your code:

- **Decorator Support**: Intuitive decorators (@workflow, @agent, @task) for automatic code instrumentation
- **Automatic Instrumentation**: Out-of-the-box support for popular third-party libraries and frameworks

### Broad Compatibility

Netra SDK supports a wide range of providers and frameworks:

- **LLM Providers**: OpenAI, Anthropic Claude, Google GenAI, Mistral AI, Aleph Alpha, AWS Bedrock, Groq, Ollama, Replicate, Together AI, Hugging Face Transformers, Vertex AI, IBM Watson X

- **AI Frameworks**: LangChain, LangGraph, LlamaIndex, Haystack, CrewAI, Model Context Protocol (MCP)
